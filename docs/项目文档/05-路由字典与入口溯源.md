# 路由字典与入口溯源

## 前端路由

### Web 端（Next.js App Router）

所有页面文件位于 `apps/web/app/`。

#### 公开路由（无需认证）

| 路径 | 页面文件 | 功能 |
|------|---------|------|
| `/` | `(landing)/page.tsx` | 首页重定向 |
| `/homepage` | `(landing)/homepage/page.tsx` | 着陆页（营销） |
| `/about` | `(landing)/about/page.tsx` | 关于页面 |
| `/changelog` | `(landing)/changelog/page.tsx` | 更新日志 |
| `/login` | `(auth)/login/page.tsx` | 登录页（Email OTP + Google OAuth） |
| `/invite/[id]` | `(auth)/invite/[id]/page.tsx` | 邀请接受页 |
| `/onboarding` | `(auth)/onboarding/page.tsx` | 首次使用引导 |
| `/auth/callback` | `auth/callback/page.tsx` | Google OAuth 回调 |

#### 受保护路由（需要认证 + 工作空间）

所有路由在 `(dashboard)/` 路由组下，由 `DashboardGuard` 统一守卫。

| 路径 | 页面文件 | 功能 |
|------|---------|------|
| `/issues` | `(dashboard)/issues/page.tsx` | Issue 列表（看板/列表视图） |
| `/issues/[id]` | `(dashboard)/issues/[id]/page.tsx` | Issue 详情 |
| `/my-issues` | `(dashboard)/my-issues/page.tsx` | 我的 Issue |
| `/projects` | `(dashboard)/projects/page.tsx` | 项目列表 |
| `/projects/[id]` | `(dashboard)/projects/[id]/page.tsx` | 项目详情 |
| `/agents` | `(dashboard)/agents/page.tsx` | Agent 管理 |
| `/autopilots` | `(dashboard)/autopilots/page.tsx` | 自动化工作流 |
| `/autopilots/[id]` | `(dashboard)/autopilots/[id]/page.tsx` | 工作流详情 |
| `/runtimes` | `(dashboard)/runtimes/page.tsx` | Runtime 管理 |
| `/skills` | `(dashboard)/skills/page.tsx` | 技能管理 |
| `/inbox` | `(dashboard)/inbox/page.tsx` | 通知收件箱 |
| `/settings` | `(dashboard)/settings/page.tsx` | 设置（多 Tab） |

#### 路由守卫链

```
请求 → Root Layout (ThemeProvider + CoreProvider)
     → (dashboard) Layout (DashboardLayout)
     → DashboardGuard
        ├→ 无用户 → 重定向 /login
        ├→ 无工作空间 → 重定向 /onboarding
        └→ 通过 → 渲染页面组件
```

### 桌面端（React Router Memory Router）

路由定义在 `apps/desktop/src/renderer/src/routes.tsx`，路由结构与 Web 端镜像。

| 路径 | 组件 | 桌面端特殊行为 |
|------|------|---------------|
| `/issues` | IssuesPage | 同 Web |
| `/issues/:id` | IssueDetailPage | 桌面端定制详情页 |
| `/projects` | ProjectsPage | 同 Web |
| `/projects/:id` | ProjectDetailPage | 桌面端定制 |
| `/autopilots` | AutopilotsPage | 同 Web |
| `/autopilots/:id` | AutopilotDetailPage | 桌面端定制 |
| `/my-issues` | MyIssuesPage | 同 Web |
| `/runtimes` | RuntimesPage | 额外注入 DaemonRuntimeCard |
| `/skills` | SkillsPage | 同 Web |
| `/agents` | AgentsPage | 同 Web |
| `/inbox` | InboxPage | 同 Web |
| `/settings` | SettingsPage | 额外注入 DaemonSettingsTab |
| `/login` | DesktopLoginPage | 桌面端专用登录页 |
| `/onboarding` | OnboardingWizard | 同 Web |
| `/invite/:id` | InvitePage | 同 Web |

**多 Tab 系统**：每个 Tab 拥有独立的 `createMemoryRouter` 实例，Tab 状态通过 Zustand Store 持久化到 localStorage。

## 后端路由

所有路由定义在 `server/cmd/server/router.go`。

### 中间件链

```
Request → RequestID → RequestLogger → Recoverer → CSP → CORS
       → [Auth Middleware (如需要)]
       → [Workspace Middleware (如需要)]
       → Handler
```

## 入口文件溯源

| 入口 | 文件路径 | 启动方式 |
|------|---------|---------|
| Web 前端 | `apps/web/app/layout.tsx` | `pnpm dev:web` (Next.js, port 3000) |
| 桌面应用 | `apps/desktop/src/renderer/src/main.tsx` | `pnpm dev:desktop` (electron-vite) |
| Go HTTP 服务 | `server/cmd/server/main.go` | `make server` (port 8080) |
| Go CLI | `server/cmd/multica/main.go` | `make cli ARGS="..."` |
| 数据库迁移 | `server/cmd/migrate/main.go` | `make migrate-up` |
| 本地 Daemon | Go 后台进程 | `make daemon` |
