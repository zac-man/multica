# 开发者指南与复用资产

### React Query Hooks（packages/core/）

所有 Query/Mutation 遵循统一的命名模式：

- **Queries**: `useQuery(useXxxQueryOptions(wsId, params))` — 由 `xxx/queries.ts` 导出 Options
- **Mutations**: `useMutation(useXxxMutationOptions())` — 由 `xxx/mutations.ts` 导出 Options

| 领域 | Query Options | Mutation Options |
|------|--------------|-----------------|
| Workspace | listWorkspaces, workspaceAgents, workspaceMembers, workspaceSkills, workspaceInvitations | createWorkspace, updateWorkspace, deleteWorkspace, createMember, updateMember, deleteMember |
| Issue | issueList, issueDetail, issueTimeline, issueReactions, issueSubscribers, issueSearch, childIssues, childProgress | createIssue, updateIssue, deleteIssue, batchUpdateIssues, batchDeleteIssues |
| Comment | commentList | createComment, updateComment, deleteComment |
| Reaction | — | addCommentReaction, removeCommentReaction, addIssueReaction, removeIssueReaction |
| Agent | — (in workspace) | createAgent, updateAgent, archiveAgent, restoreAgent |
| Skill | — (in workspace) | createSkill, updateSkill, deleteSkill, importSkill |
| Project | projectList, projectDetail, projectSearch | createProject, updateProject, deleteProject |
| Chat | chatSessions, chatMessages, pendingTask, pendingTasks | sendMessage, createSession, archiveSession, markChatRead |
| Inbox | inboxList, unreadCount | markRead, archive, markAllRead, archiveAll |
| Autopilot | autopilotList, autopilotDetail, autopilotRuns, autopilotTriggers | createAutopilot, updateAutopilot, deleteAutopilot, triggerAutopilot, createTrigger, updateTrigger, deleteTrigger |
| Runtime | runtimeList, runtimeUsage, runtimeActivity | deleteRuntime, initiatePing, initiateUpdate |
| Pin | pinList | createPin, deletePin, reorderPins |

### 平台抽象接口

#### NavigationAdapter

```typescript
// packages/views/navigation/types.ts
interface NavigationAdapter {
  push(path: string): void;
  replace(path: string): void;
  back(): void;
  pathname: string;
  searchParams: URLSearchParams;
  openInNewTab?: (path: string, title?: string) => void;
  getShareableUrl?: (path: string) => string;
}
```

使用方式：

```tsx
import { useNavigation } from '@multica/views/navigation/context';
const nav = useNavigation();
nav.push('/issues/123');
```

#### StorageAdapter

```typescript
// packages/core/platform/storage.ts
interface StorageAdapter {
  getItem(key: string): string | null;
  setItem(key: string, value: string): void;
  removeItem(key: string): void;
}
```

默认提供 SSR 安全的 localStorage 包装。

## 后端可复用资产

### 数据库查询（sqlc Generated）

所有查询方法在 `server/pkg/db/generated/` 中自动生成。

**使用模式**：

```go
// Handler 中调用
issue, err := h.Queries.GetIssueInWorkspace(ctx, db.GetIssueInWorkspaceParams{
    ID:          issueID,
    WorkspaceID: workspaceID,
})
```

### 事件总线

```go
// 发布事件
h.Bus.Publish(ctx, events.Event{
    Type: protocol.EventIssueCreated,
    Payload: payload,
    WorkspaceID: wsID,
    ActorID: userID,
})

// 订阅事件
h.Bus.Subscribe(protocol.EventIssueCreated, func(ctx context.Context, e events.Event) {
    // 处理逻辑
})
```

### WebSocket Hub

```go
// 广播到工作空间
h.Hub.BroadcastToWorkspace(workspaceID, protocol.Message{...})

// 发送给特定用户
h.Hub.SendToUser(userID, protocol.Message{...}, excludeWorkspace...)

// 全局广播
h.Hub.Broadcast(protocol.Message{...})
```

### 存储抽象

```go
// 文件上传（自动选择 S3 或本地）
url, err := h.Storage.Upload(ctx, filename, contentType, reader)
```

## 核心数据模型

### Issue（最核心的实体）

```typescript
interface Issue {
  id: string;
  workspaceId: string;
  number: number;          // 工作空间内自增编号
  title: string;
  description: string;     // TipTap JSON
  status: IssueStatus;     // backlog | todo | in_progress | done | cancelled | (其他)
  priority: IssuePriority; // none | urgent | high | medium | low
  assigneeType: 'member' | 'agent' | null;
  assigneeId: string | null;
  creatorType: 'member' | 'agent';
  creatorId: string;
  parentIssueId: string | null;  // 子 Issue
  projectId: string | null;
  position: number;        // 看板排序
  dueDate: string | null;
  originType: 'manual' | 'autopilot' | null;
  labels: IssueLabel[];
}
```

### Agent

```typescript
interface Agent {
  id: string;
  workspaceId: string;
  name: string;
  runtimeId: string;
  runtimeMode: 'local' | 'cloud';
  runtimeConfig: Record<string, any>;  // JSON
  instructions: string;       // System prompt
  visibility: 'workspace' | 'private';
  status: 'idle' | 'working' | 'offline';
  maxConcurrentTasks: number; // 默认 6
  ownerId: string;
  customEnv: Record<string, string>;  // 环境变量
  skills: Skill[];
}
```

### Workspace

```typescript
interface Workspace {
  id: string;
  name: string;
  slug: string;
  description: string;
  context: string;            // AI 上下文信息
  settings: Record<string, any>;  // JSONB
  repos: RepoConfig[];        // 关联仓库
  issuePrefix: string;        // 如 "MUL"
  members: Member[];
}
```
