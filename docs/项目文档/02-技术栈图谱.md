# 技术栈图谱

## 前端技术栈

### 核心框架

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 19.2.3 | UI 渲染引擎 |
| Next.js | 16.2.3 | Web 应用框架（App Router） |
| Electron | 39.2.6 | 桌面应用框架 |
| electron-vite | 5.x | Electron 构建工具 |
| React Router DOM | 7.6.0 | 桌面端路由（Memory Router，每 Tab 独立） |
| TypeScript | 5.9.x | 类型系统 |

### 工程化

| 技术 | 用途 |
|------|------|
| pnpm 10.28.2 + workspace | Monorepo 包管理 |
| Turborepo | 构建编排，依赖感知任务图 |
| Vitest | 单元测试 |
| Playwright | E2E 测试 |
| ESLint | 代码规范 |
| shadcn/ui (Base UI) | 组件库（`@base-ui/react`，非 Radix） |

### 状态管理

| 技术 | 用途 |
|------|------|
| Zustand 5.x | 客户端状态（UI 选择、筛选、草稿、面板状态） |
| TanStack React Query 5.x | 服务端状态（Issue、Agent、Workspace 等所有 API 数据） |

### UI 与交互

| 技术 | 用途 |
|------|------|
| Tailwind CSS 4 | 样式系统 |
| TipTap | 富文本编辑器（Issue 描述、评论） |
| dnd-kit | 拖拽排序（看板卡片、Tab 等） |
| cmdk | 命令面板（全局搜索） |
| emoji-mart | Emoji 选择器 |
| recharts | 图表（Runtime 用量统计） |
| shiki | 代码语法高亮 |
| react-markdown | Markdown 渲染 |

### 实时通信

| 技术 | 用途 |
|------|------|
| 原生 WebSocket | 实时事件推送（浏览器 API，非额外库） |

## 后端技术栈

### 核心框架

| 技术 | 版本 | 用途 |
|------|------|------|
| Go | 1.26.1 | 后端主语言 |
| Chi | v5 | HTTP 路由与中间件 |
| gorilla/websocket | v1 | WebSocket 连接管理 |
| sqlc | - | 类型安全的 SQL 代码生成 |
| Cobra | - | CLI 命令行框架 |

### 数据库

| 技术 | 用途 |
|------|------|
| PostgreSQL 17 | 主数据库 |
| pgvector | 向量搜索扩展（PostgreSQL 插件） |
| pgx/v5 | Go PostgreSQL 驱动 |

### 存储

| 技术 | 用途 |
|------|------|
| AWS S3 | 文件存储（生产环境） |
| Local Filesystem | 文件存储（开发环境） |
| CloudFront | CDN 签名访问 |

### 认证

| 技术 | 用途 |
|------|------|
| JWT (HMAC-SHA256) | 用户会话令牌（30 天过期） |
| PAT (`mul_` 前缀) | API 令牌（哈希存储） |
| HttpOnly Cookie | Web 端安全令牌传输 |
| Google OAuth 2.0 | 第三方登录 |
| Email OTP | 邮箱验证码登录 |

### 基础设施

| 技术 | 用途 |
|------|------|
| Docker / Docker Compose | 容器化部署 |
| GitHub Actions | CI/CD |
| GoReleaser | CLI 多平台构建发布 |
| Homebrew Tap | CLI 分发 |
