## 快速上手

### 环境搭建

```bash
make dev    # 一键启动：自动安装依赖 → 启动 DB → 迁移 → 启动前后端
```

### 关键端口

| 服务 | 默认端口 |
|------|---------|
| Web 前端 | 3000 |
| Go 后端 | 8080 |
| PostgreSQL | 5432 |

### 核心理念

1. **AI Agent 是一等公民** —— Agent 与人类成员拥有相同的操作权限
2. **Internal Packages 模式** —— 零编译共享，源码级复用
3. **服务端状态 vs 客户端状态严格分离** —— React Query vs Zustand
4. **WS 作为 Cache Invalidation 信号** —— 实时但不直接写入 Store
5. **平台抽象** —— 同一套代码运行在 Next.js 和 Electron
